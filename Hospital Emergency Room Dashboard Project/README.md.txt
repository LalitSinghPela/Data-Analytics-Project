📊 Hospital Emergency Room Analytics Dashboard

🔹 Project Overview

This project analyzes hospital emergency room patient data using Power BI and Excel to generate meaningful insights and interactive visualizations.

🔹 Dataset Information 
Total Records: 9,216+ 
Data Cleaning: Excel
Visualization Tool: Power BI

🔹 Key Insights

Total Patients Analysis Average Wait Time Satisfaction Score Analysis Gender Distribution Department-wise Patient Visits Day-wise Patient Traffic Interactive Filters and Slicers

🔹 Tools Used

Power BI Excel Data Cleaning Data Visualization
